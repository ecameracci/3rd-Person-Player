# 3rd Person Camera Template

This is a basic Godot project with a 3rd person player setup that can be controlled by either the mouse or a controller.

## Controls
### Gamepad
Left Stick - Movement 

Right Stick - Rotate camera 

South face button - Jump 

### Keyboard + Mouse
W,A,S,D - Movement

Mouse - Rotate camera

Space button - Jump
